# Akıllı Bellek Yönetimi Uygulaması

Bu proje, çalışan programların RAM kullanımını analiz etmeyi ve gereksiz bellek tüketimini tespit etmeyi amaçlayan bir uygulama geliştirmek için hazırlanmıştır. 

Uygulama sayesinde sistemde çalışan süreçlerin bellek kullanımı incelenebilecek ve yüksek bellek tüketimi olan durumlar analiz edilebilecektir. Projede ayrıca elde edilen verilerin kullanıcıya anlaşılır bir şekilde sunulması hedeflenmektedir.

## Projenin Amaçları
- Çalışan programların RAM kullanımını analiz etmek
- Gereksiz veya yüksek bellek tüketimini tespit etmek
- Analiz sonuçlarını raporlamak
- Kullanıcıya basit bir arayüz üzerinden bilgi sunmak

## Kullanılacak Teknolojiler
- C++
- Python
- Linux
- Valgrind
